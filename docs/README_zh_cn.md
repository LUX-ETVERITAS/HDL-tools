# HDL Tools Bucket

![License](https://img.shields.io/badge/license-MIT-yellow)
![Documentation](https://img.shields.io/badge/documentation-yes-brightgreen)

hdl-tools-bucket 是一个面向 HDL 开发的 Scoop 软件仓库，收录与 Verilog、SystemVerilog 以及 FPGA 开发流程相关的工具集合。

本仓库主要包括：

FPGA 开发工具链

Verilog / SystemVerilog 仿真器与编译器

综合与实现工具

开源 EDA 工具

一些官方 bucket 未收录的工具或旧版本

偏实验性或研究方向的 HDL 工具

本仓库旨在作为 Scoop 官方仓库（main、extras、versions）的补充，为硬件工程师、FPGA 开发者和数字电路研究者提供更完整的工具支持。
