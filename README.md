hdl-tools-bucket is a Scoop bucket that provides a curated collection of tools for HDL development, including Verilog, SystemVerilog, and FPGA workflows.

This repository focuses on:

FPGA development toolchains

Verilog / SystemVerilog simulators and compilers

Synthesis and implementation tools

Open-source EDA utilities

Legacy or hard-to-find HDL-related software

Experimental and research-oriented tooling

The goal of this bucket is to complement the official Scoop buckets (main, extras, versions) by providing specialized tools for hardware engineers, FPGA developers, and digital design researchers.
